# Лабораторна робота №1. Фронтенд
## Варіант №9
**Тема:** Заявки в технічну підтримку для студентів  
**Ключова сутність:** Заявка

## Функціонал
* Створення нових заявок до технічної підтримки.
* Відображення списку заявок у вигляді таблиці.
* Редагування інформації про заявку.
* Видалення заявок.
* Вибір статусу та пріоритету заявки.
* Перевірка обов’язкових полів форми.
* Збереження даних у localStorage з автоматичним відновленням після перезавантаження сторінки.
